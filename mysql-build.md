# mysql-buildメモ

DATADIRを何も設定しないと、DATADIRは以下になる。
`/etc/my.cnf`以外に設置する場合は、以下を参考にする。

チェックコマンド

```sql
show variables like "datadir";
```

## mysql 4.0.30

```sql
+---------------+---------------------------------+
| Variable_name | Value                           |
+---------------+---------------------------------+
| datadir       | /home/vagrant/mysql/4.0.30/var/ |
+---------------+---------------------------------+
```

## mysql 4.1.25

```sql
+---------------+---------------------------------+
| Variable_name | Value                           |
+---------------+---------------------------------+

+---------------+---------------------------------+
```
